# settle

https://settlegame.herokuapp.com/

Online tile-based board game inspired by [Carcassonne](https://en.wikipedia.org/wiki/Carcassonne_(board_game)).

## Development

### Basic

``` bash
# install dependencies
yarn install

# serve with hot reload at http://localhost:8080/
yarn run dev
```

### Testing
``` bash
# run unit tests
yarn run unit

# run api tests
yarn run api

# run all tests
yarn test
```
